# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you find a security vulnerability in VerifyIn, please opening a GitHub issue at https://github.com/Kiana-F/verifyin/issues with:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

You'll receive a response within 48 hours. If confirmed, we'll aim to release a fix within 7 days and credit you in the release notes (unless you prefer to remain anonymous).

Thank you for helping keep VerifyIn secure.
