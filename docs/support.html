Common questions
Is VerifyIn really free? +
Yes — completely free, forever. MIT license means you can use it, modify it, and even build a business on top of it. No hidden fees, no "free tier" limits.
Do I need technical skills to set it up? +
You need basic comfort with a terminal. The setup script handles everything — just fill in your API keys (Africa's Talking, SendGrid, Cloudinary) and run one command.
Does it work without internet? +
The security guard console works offline thanks to the service worker — it caches the app shell. API calls (check-in, check-out) still need a connection.
Can I contribute code? +
Absolutely — that's the whole point of open source! See CONTRIBUTING.md for how to get started. All skill levels welcome.
Why PayPal and not M-Pesa? +
PayPal works internationally. M-Pesa integration is on the roadmap — when it's done, we'll add a Paybill option here too!
